- 差别对比
  - 顶层分类基本一致
  - 脑图中按照维度组织知识点，有助于理解复杂的知识体系
    - JavaScript 知识分为静态词法、语法结构，和动态的运行时，相比教程式学习是站在一个较高的视角
  - 自己所列目录考虑学习进程，考虑了按入门与提高组织内容，不同于脑图(目标是全面覆盖知识体系)

    - 除浏览器外，其他前端平台：Node, Electron, 小程序
    - 浏览器 DOM API 中 Range 概念
    - Javascript 内置类型与内置对象(类)的区别
